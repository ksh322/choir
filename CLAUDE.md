## 사이트 구조

```
app/page.tsx         ← 메인 페이지 (컴포넌트 조합)
components/
  hero.tsx           ← 타이틀 / 성가대 소개 문구
  about.tsx          ← 지휘자 및 성가대 배경 소개
  skills.tsx         ← 성악·건반·지휘·교육 역량
  experience.tsx     ← 단원 명단 및 역할
  projects.tsx       ← 공연·연주회 이력
  contact.tsx        ← 문의 폼 및 연락처
  navigation.tsx     ← 좌측 고정 사이드바 네비게이션
```

## 홍보 방향 가이드

### 핵심 메시지
> "음악을 통해 하나님의 영광을 드러내고, 성가대원들과 함께 아름다운 하모니를 만들어갑니다."

### 콘텐츠 우선순위
1. 공연 일정 및 공지 → `components/projects.tsx`
2. 단원 모집 문구 → `components/hero.tsx` 또는 `components/contact.tsx`
3. 성가대 역량 강조 → `components/skills.tsx`
4. 지역 행사·협연 홍보 → `components/projects.tsx`에 추가

### 작업 시 주의사항
- 모든 텍스트는 **한국어** 기준으로 작성하고, 영문 병기는 선택 사항
- 성가대원 명단 변경은 `components/experience.tsx`에서만 수정
- 이미지 파일은 `public/` 폴더에 저장 (공연 사진, 배너 등)
- 세례명은 단원 개인 동의 후 게시
